# 权重与置信度 v1.0

## 通用 100 分制

| 维度 | 分值 | 说明 |
| --- | ---: | --- |
| 规则命中 | 35 | 命中已确定的四有卷别归入资料、硬边界或项目 Excel 规则 |
| 关键字段命中 | 20 | 文号、日期、单位、题名、文保点、文件类型等支持该卷 |
| 正文/OCR关键词 | 15 | OCR 或抽取文本中出现明确关键词 |
| RAG 相似度 | 10 | 与已确认样例、规则说明、同类材料语义相似 |
| 来源路径辅助 | 10 | 文件夹名、年度目录、类别目录提供辅助信号 |
| 文件格式匹配 | 5 | 格式符合该卷材料形态，如图片候选照片卷 |
| OCR/抽取质量 | 5 | 文本清晰、置信度高、证据页完整 |

## 硬性处理

- 明确规则冲突时，以规则优先，不用总分强行通过。
- OCR 质量低、字段缺失严重、多个候选卷别分差小于 10 分，标记待核。
- 来源路径只能辅助，不得单独决定卷别。
- RAG 相似度只作为召回和解释，不得压过明确规则。
- 图片格式不符合闸门时，不自动归入 `1201` 或 `1301`。

## 置信等级

| 等级 | 建议条件 |
| --- | --- |
| 高 | 总分 >= 80，且规则命中或关键字段命中明确 |
| 中 | 总分 60-79，证据基本成立但需人工看一眼 |
| 低 | 总分 40-59，只有弱关键词、路径或相似度支持 |
| 待核 | 总分 < 40，或触发硬性待核条件 |

## 人工权重调整

权重应写入可调整配置，例如 `05_classify/weights.json`。调整后必须在日志中记录：

- 调整前后权重。
- 调整原因。
- 影响的候选卷别数量。
- 是否经过人工抽样复核。
