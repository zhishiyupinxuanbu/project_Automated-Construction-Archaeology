# 第二章第二节遗址概述与价值评估规则 v0.2

本文件用于固定第二章 `（二）【文物名称】概述` 的写作规则。价值陈述和价值评估不再另设独立第三节，而是作为第二节下的 `5.价值陈述` 和 `6.价值评估` 保留。本节遗址情况一般来自文物调查回函、文物调查报告、文物核查报告或主管部门函件。

## 第二章第二节固定小节

第二章第二节标题随项目涉及文物替换，固定格式为 `（二）【文物名称】概述`。其下小节标题固定为：

- `1.遗址基本情况`
- `2.遗址现状评估`
- `3.调查、发掘、保护工程情况`
- `4.保护范围及建设控制地带`
- `5.价值陈述`
- `6.价值评估`

未从文物调查回函或调查类材料中抽取到具体正文时，先判断是否属于必需内容；必需内容作为阻断缺项处理，非必需细节略过，不得在正文标注待补充，不得合并为一个总段。

`5.价值陈述` 不等同于资料摘录缺项。事实抽取模块应先抽取文物级别、时代、类型、保存现状、公布情况、历史沿革、既有调查和价值线索；报告拼装阶段再按文化遗产价值维度组织成段文字。常用维度包括历史价值、文化价值、社会价值、军事价值、景观风貌价值等，具体采用哪些维度以文物类型和材料事实为准。

写作要求：

1. 每个适用价值维度原则上独立成段，不写成一句话清单。
2. 每段必须回扣该文物的具体事实，如时代、形制、位置、历史背景、保存现状、区域文化或军事防御体系关系。
3. 不把其他项目、其他文物的价值结论直接迁移过来。
4. 不把“资料不足”写进正文；必需事实不足时先阻断，事实够但价值表述需综合时由 AI 按维度撰写。

## 保护区划术语

写作时必须以文物相关材料的正式表述为准，不统一改写保护区划术语。

常见术语包括但不限于：

- 保护范围。
- 建设控制地带。
- 遗产区。
- 缓冲区。
- 生态景观规划保护区。
- 本体范围。
- 保护区划。

使用规则：

1. 文物材料写“保护范围、建设控制地带”，报告中就使用“保护范围、建设控制地带”。
2. 世界遗产或遗产类材料写“遗产区、缓冲区”，报告中就使用“遗产区、缓冲区”。
3. 如果材料同时出现多套区划术语，必须分别写清，不合并成一个泛称。
4. 如果材料未明确保护区划名称且该信息为必需内容，作为阻断缺项处理；非必需细节略过，不要擅自补成“保护范围/建设控制地带”，也不写待核验提示。
5. 保护区划属于高风险字段，内部抽取表必须记录来源文件；正式正文不写“根据【来源文件】”类项目资料出处引导语。

推荐句式：

> 【文物名称】为【文物级别/类型/时代】，位于【位置】。其【材料原文术语一】为【对应内容】，【材料原文术语二】为【对应内容】。文物保存现状为【保存现状】，主要价值体现在【历史/科学/艺术/社会/景观/其他价值】。

## 价值评估

`6.价值评估` 中，从小节标题后的价值评估准则说明段到 `表1 遗产价值分级量表` 及其表格内容均为完全固定文本。该固定段不得摘要、改写、删表或替换为待补提示；本项目文物价值等级和理由只能写在固定表格之后。

固定说明段为：

```text
《世界文化遗产影响评估指南》中的价值评估准则指出，对文物开展文物影响评估时，应考量其具备的文物价值、当地价值或国家价值，以及国家研究规划所明确的优先顺序与建议。同时，还需考虑文物所体现出的价值。本次文物影响评估报告依据《世界文化遗产影响评估指南》中的遗产价值分级量表（如下图所示），针对项目涉及的不可移动文物开展文物价值评估。
```

固定表名为：

```text
表1 遗产价值分级量表
```

该量表不只适用于世界遗产，也可用于各类遗产资产的价值评级。

表格后的评价段仅作变量替换，固定句式为：

```text
根据上述文物价值陈述，结合遗产价值分级量表，【文物名称】属于已公布为保护单位的建筑物（【文物保护单位级别】），具备【价值方面】。因此评估【文物名称】的遗产价值分级为“【等级】”。
```

只替换【文物名称】、【文物保护单位级别】、【价值方面】、【等级】四类信息。不得改写成“根据现有资料……”，不得加入时代、类型、山险、线性遗存特征、待专业负责人确认等扩写内容。

使用规则：

1. 先从项目资料中提取文物级别、类型、时代、公布情况、保存现状、价值说明和研究意义。
2. 再判断该文物主要对应量表中的哪一类资产：考古学、建成遗址或历史城市景观、历史景观、非物质文化遗产或相关内容。
3. 按文物资料中的级别和价值，在量表中对应其价值评级：非常高、高、中、低、可忽略或未知。
4. 评级理由必须写清“为什么对应这一档”，不能只写结论。
5. 不因量表名称含“世界文化遗产”就限定为世界遗产项目使用。

常见对应关系提示：

- 世界遗产及反映突出普遍价值的属性，通常对应“非常高”。
- 全国重点文物保护单位、具有国家级价值的遗产资产，通常对应“高”。
- 已公布为保护单位且具有区域或地方重要价值的文物，可根据材料对应“中”或“低”；自治区级、市县级文物保护单位不能机械套档，需结合量表和价值说明判断。
- 价值信息不足且价值评级为必需内容时先阻断；非必需细节略过，不写价值资料待补充提示。

## 固定表格

固定表格放在 `assets/12-遗产价值分级量表模板.md`。写作第二章第二节 `6.价值评估` 时，必须在固定说明段和固定表名之后插入该表，并在表后按固定评价段替换变量。
