# 文物影响评估事实抽取字段表 v0.1

本文件规定从项目资料夹中抽取哪些事实，以及每类事实的优先来源。所有事实必须在内部抽取表中保留来源文件名或路径；无法确认的信息进入内部缺口或风险记录，不得以【待补充】【待确认】【待复核】等占位形式进入正式正文。

## 项目基本事实

| 字段 | 优先来源 | 风险要求 |
| --- | --- | --- |
| 项目名称 | 合同、项目资料、报告、函件、规划图标题 | 多文件不一致时列出差异，不擅自统一 |
| 建设单位 | 合同、请示、项目说明、审查材料 | 必须有文件依据 |
| 项目位置 | 项目说明、规划图、调查报告、坐标/KML | 行政区划与具体地点分开写 |
| 建设内容 | 项目说明、规划总平面图、设计资料、合同服务范围 | 区分拟建、改建、配套设施、运营内容 |
| 建设规模 | 规划图、设计资料、项目说明、坐标表 | 面积、高度、建筑数量等高风险字段需逐项标来源 |
| 建设阶段 | 请示、合同、执行资料、报告日期 | 不明确时进入内部缺口记录；必需时阻断，非必需时正文略过 |
| 建设必要性原文章节 | 可研报告、可行性研究报告、建设方案、项目说明 | 完整保留分条理由和论证展开，不用 AI 概述替代 |
| 建设及运营方案原文章节 | 可研报告、可行性研究报告、建设方案、运营方案 | 定位建设规模、技术指标、运营方案、运营规模、安全保障等章节 |
| 选址分析原文章节 | 可研报告、可行性研究报告、建设方案 | 定位选址分析、选址必要性分析或方案比选章节，供第四章完整承接 |

## 文物基本事实

| 字段 | 优先来源 | 风险要求 |
| --- | --- | --- |
| 文物名称 | 文物调查报告、文物部门函件、文保单位资料 | 名称必须与正式文件一致 |
| 文物级别 | 公布文件、调查报告、文物部门函件 | 不凭常识判断级别 |
| 文物类型与时代 | 调查报告、文保单位资料、考古报告 | 缺失时进入内部缺口记录；必需时阻断，非必需时正文略过 |
| 保存现状 | 调查报告、现场照片、勘探报告 | 区分本体、周边环境、地下遗存 |
| 既有调查发掘保护史 | 文物调查报告、文保单位资料、三普/四普资料、考古发掘资料、保护工程资料、公布文件 | 写既有文物工作史，不写本项目调查勘探工作 |
| 保护区划术语 | 公布文件、文物部门函件、调查报告、保护规划、世界遗产材料 | 必须跟随材料原文，如保护范围、建设控制地带、遗产区、缓冲区、本体范围等，不擅自统一改写 |
| 保护范围/建设控制地带/遗产区/缓冲区等具体内容 | 公布文件、文物部门函件、调查报告、保护规划、世界遗产材料 | 高风险字段，必须有依据 |
| 价值特征 | 调查报告、文保单位资料、成稿样本的表达规则 | 不能用其他项目价值替代；事实抽取不足时标记需按历史、文化、社会、军事、景观风貌等适用维度成段撰写 |
| 遗产价值评级 | 文物级别、类型、价值说明、保存现状、《世界文化遗产影响评估指南》遗产价值分级量表 | 专业复核字段，必须说明对应量表的理由 |

## 空间关系事实

| 字段 | 优先来源 | 风险要求 |
| --- | --- | --- |
| 项目红线 | 红线图、坐标表、KML/OVKML、用地范围图 | 必须写明图件或坐标来源 |
| 文保点坐标 | 文物调查报告、文保单位资料、文物部门函件、保护规划、长城资源调查材料 | 高风险字段，内部抽取表必须写明来源；坐标系不明时进入内部缺口记录，必需时阻断 |
| 项目与文物形态 | 项目图件、红线、文物资料、保护区划图 | 判断点状、线状、面状关系，决定第四章写法 |
| 最近距离 | 叠合图、距离图、调查报告、GIS/KML 量算结果 | 无量算依据时进入内部缺口记录；必需时阻断，非必需时正文略过 |
| 是否涉及保护范围 | 叠合图、文物部门函件、调查报告 | 不可凭肉眼估计下结论 |
| 是否涉及建设控制地带 | 叠合图、文物部门函件、调查报告 | 不可凭肉眼估计下结论 |
| 重叠面积 | 坐标量算、调查报告、GIS 输出 | 无明确数据时进入内部缺口记录；必需时阻断，非必需时正文略过 |
| 选址比选 | 项目说明、规划材料、成稿/评审材料 | 没有多方案材料时不要虚构比选 |

## 影响识别事实

| 字段 | 优先来源 | 风险要求 |
| --- | --- | --- |
| 设计控制要素 | 规划图、设计资料、建筑高度、色彩、材料、功能布局 | 影响景观风貌时必须有设计事实 |
| 预填表驱动条文 | 第五章人工预填表指定或暗示的法规、保护规划、管理办法、主管部门文件 | 先抽项目事实，再按预填表需求定向抽具体条文或控制要求；只输出事实与条文，不输出合规结论 |
| 建设期活动 | 施工说明、勘探报告、工程资料、常规施工逻辑 | 常规推断必须标为推断，不当作已提供事实 |
| 运营期活动 | 项目说明、运营方案、功能定位、游客/使用人群材料 | 无运营方案时进入内部缺口记录；必需时阻断，非必需时正文略过 |
| 地下文物风险 | 考古调查勘探报告、验收意见、探孔/剖线材料 | 不擅自得出“无地下文物风险”结论 |
| 勘探工作 | 考古调查、勘探工作报告 | 第四章第三节只提取项目名称、勘探公司和勘探结论并套入固定文本；实际完成面积按报告原文填入固定句；不提取方法、探孔/剖线、验收过程入正文，不插图 |
| 专家或部门意见 | 评审意见表、文物部门函件、会议纪要 | 必须逐条保留来源 |

## 正式文件元信息

| 字段 | 优先来源 | 风险要求 |
| --- | --- | --- |
| 文件题名 | 红头文件首页、正文标题、文件属性 | 与正文标题不一致时记录冲突 |
| 发文单位 | 红头、落款、印章、文件属性 | 不凭文件夹名补写 |
| 文号 | 红头文号、正文引用、落款附近信息 | 必须完整识别；不准或不全时进入内部问题 |
| 发文日期 | 落款、红头日期、文件属性 | 多日期并存时说明日期类型 |
| 正文页范围 | PDF 页码、图片页序、OCR 分页 | 第三章支持性文件插图优先使用完整正文页 |
| 附件页范围 | 附件标题、附表/附图页、页码 | 附件页默认不替代正文页 |

## 抽取输出要求

- 每条事实在内部抽取表中标注“来源：文件名/页码或章节/表图号”，页码无法识别时至少标文件名；正式正文承接项目资料时不写“根据/依据XX报告”等来源引导语。
- 同一字段出现多个版本时，列入内部冲突记录，不要自行选择；若为正文必需事实，先阻断并提示用户。
- 影响等级、审批结论、是否可实施必须进入人工审核点，未能确定时不得以“待专业负责人确认”等表述写入正式正文。
