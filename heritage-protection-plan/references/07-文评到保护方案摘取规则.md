# 文评到保护方案摘取规则

保护方案与文评报告“差不多”，但不是复制文评全文。正确做法是从文评中摘取可复用事实、空间关系、影响问题和措施建议，再改写为保护方案的实施性章节。

## 必读文评章节

优先读取文评报告以下章节：

1. 目录：确认文评结构和可摘取章节。
2. 总则：摘取编制背景、依据、原则；评估目标改为保护方案编制目的。
3. 文物概况：摘取文物名称、级别、类型、保存现状、保护范围和价值信息。
4. 项目概况：摘取项目名称、建设单位、地点、建设内容、施工及运营方案。
5. 空间关系：摘取项目与文物的相对位置、保护范围/建设控制地带关系、勘探情况。
6. 影响分析：摘取风险来源和影响路径，不照搬最终影响等级。
7. 减缓措施建议：摘取可执行措施，补成责任主体、阶段、巡查、监测、应急和附件。
8. 结论及建议：只摘取需要落实的建议，不照搬“结论”作保护方案结论。

## 章节映射

| 文评内容 | 保护方案位置 | 改写要求 |
|---|---|---|
| 编制背景 | 一、总则-编制背景 | 从“开展影响评估”改为“制定保护方案、落实保护要求” |
| 评估目标/评估内容 | 一、总则-编制目的/实施纲要 | 删除评估方法表述，改成保护责任和措施落实 |
| 评估原则 | 一、总则-保护原则 | 保留最小干预、真实性、完整性、协调发展等原则 |
| 文物所在区域和文物概述 | 三、文物概况 | 精简价值评估长论证，保留保护管理所需事实 |
| 项目规划概况 | 二、项目基本情况 | 保留工程内容、施工和运营方式，不扩写商业论证 |
| 空间分布关系 | 三、文物概况或四、措施前置说明 | 用于说明为什么采取措施，不重复完整评估 |
| 设计合规性评估 | 四、保护措施 | 改写为设计控制要求和待确认事项 |
| 建设期影响评估 | 四、保护措施 | 改写为施工扰动、机械、扬尘、污水、人员管理措施 |
| 运营期影响评估 | 四、保护措施 | 改写为运营期巡查、人员活动、环境长期管控 |
| 减缓措施建议 | 四、保护措施 | 改成可执行条款，补责任主体和记录方式 |
| 评估结论 | 不直接入文 | 只保留经确认的管理建议和后续工作要求 |

## 语言改写

- “可能产生影响”改为“应采取……措施进行控制”。
- “建议采取”改为“建设单位、施工单位应……”。
- “影响较小/可接受”改为“在落实本方案措施并经专业负责人确认后，降低相关风险”。
- “应进一步核实”保留为【待专业负责人确认】或【需补充资料】。
- “评估认为”改为“本方案依据……提出……”。

## 不能摘取的内容

- 不直接摘取文评的最终影响等级。
- 不直接摘取未经核实的距离、坐标、面积和保护边界。
- 不直接摘取文评里的专家意见为事实。
- 不把文评中的“建议”写成已落实事实。

## 摘取后必须补齐

文评通常缺少以下保护方案成稿要素，写保护方案时必须补：

- 实施纲要。
- 责任主体。
- 施工前交底、培训和防护设施。
- 施工期巡查、监测和异常停工机制。
- 运营期长期管理。
- 文物安全应急预案。
- 承诺书、作业机械管理制度、巡查记录表等附件。
