## 小型公路保护方案章法

适用于单一线路、文物对象较少、措施集中在施工和运营管控的项目。

```text
一、总则
  （一）编制背景
  （二）编制目的
  （三）保护原则
  （四）编制依据
  （五）实施纲要
二、项目基本情况
  （一）项目名称及概况
  （二）项目建设的必要性
三、文物概况
四、文物保护措施
  （一）文物安全应急预案
  （二）施工人员规范化管理
  （三）施工期管控措施
  （四）运营期保护措施
附件：应急预案、承诺书、作业机械管理制度、巡查记录表
```

## 煤矿/采矿保护方案章法

适用于矿权延续、井田范围与遗址重叠、需要考古发掘前置或保护煤柱的项目。

```text
一、总则
二、项目基本情况
  （一）项目名称及概况
  （二）主要建设内容
  （三）项目建设的必要性
三、文物概况
  （一）文物对象一概述
  （二）文物对象二概述
四、文物保护措施
  （一）实施考古发掘前置
  （二）发掘前保护措施
  （三）文物安全应急预案
  （四）工作人员规范化管理
五、资金保障
附件：承诺书、作业机械管理制度、巡查记录表
```

## 建筑改扩建保护方案章法

适用于建设控制地带、景观风貌、施工扰动、运营活动管理较突出的项目。

```text
一、总则
二、项目基本情况
三、文物概况
四、文物保护措施
  （一）文物安全应急预案
  （二）景观风貌保护与设计控制措施
  （三）施工扰动与环境污染控制措施
  （四）运营期人员活动与环境管理措施
  （五）跟踪监测与整改闭环措施
附件：应急预案、承诺书、作业机械管理制度、巡查记录表
```

## 评审后复杂方案章法

适用于专家意见要求细化安全距离、保护煤柱、振动监测、施工期和运营期分别论述的项目。

```text
一、总则
二、项目基本情况
三、文物概况
四、建设项目施工期拟采用的文物保护措施
  （一）工程防护措施
  （二）环境保护措施
  （三）施工人员规范化管理
五、建设项目运营期拟采用的文物保护措施
  （一）工程防护措施
  （二）改善周边环境措施
  （三）工作人员规范化管理
六、责权清晰，制定文物安全应急预案
附件或附表：坐标表、监测记录、巡查记录、承诺书
```

## 选择规则

- 有配套文评报告时，先从文评报告摘取事实和影响问题，再套用最接近的保护方案章法。
- 无配套文评报告但有既有保护方案时，按既有保护方案章法续写或修改。
- 多文物对象、煤矿、评审意见较多时，优先使用复杂方案章法。
- 不确定项目类型时，先输出两套候选目录让用户选，不要直接锁死结构。
