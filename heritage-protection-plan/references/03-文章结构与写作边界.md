# 文章结构与写作边界

## 标准结构

保护方案的标准结构从文评报告摘取并重排。文评报告通常按“总则、文物概况、项目概况、空间关系、影响评估、减缓措施、结论”组织；保护方案应改为“总则、项目基本情况、文物概况、保护措施、保障与附件”。

适用于中大型项目、文物对象较多或已有评审意见的方案：

1. 总则
   - 编制背景
   - 编制目的
   - 保护原则
   - 编制依据
2. 项目基本情况
   - 项目名称及概况
   - 主要建设内容
   - 项目建设必要性
3. 文物概况
   - 分文物对象概述
   - 保护范围、建设控制地带和保存现状
   - 与项目空间关系
4. 文物保护措施
   - 可按施工前、施工期、运营期拆分
   - 可按工程防护、环境保护、人员管理、应急预案拆分
5. 资金保障或责任保障
6. 附表、附件

## 简化结构

适用于小型项目或只有单一文物对象、单一工程行为的方案：

1. 总则
2. 项目基本情况
3. 文物概况
4. 文物保护措施
5. 附件

## 从文评章节到保护方案章节

| 文评报告章节 | 保护方案章节 | 处理方式 |
|---|---|---|
| 总则 | 总则 | 改“评估目标/评估内容”为“编制目的/实施纲要” |
| 建设项目涉及文物概况 | 文物概况 | 摘取文物位置、保护级别、保存现状和价值信息 |
| 建设项目规划概况 | 项目基本情况 | 摘取项目名称、建设内容、必要性、施工和运营方案 |
| 项目用地范围与文物空间分布关系 | 文物概况或保护措施前置说明 | 摘取空间关系，避免保留完整评估论证 |
| 可能影响分析与评估 | 文物保护措施 | 把影响问题改写为风险来源和控制目标 |
| 减缓措施建议 | 文物保护措施 | 改写为施工前、施工期、运营期和应急落实措施 |
| 评估结论及建议 | 总则或保护措施结尾 | 只保留需落实的建议，不直接搬运结论 |

## 正文写法

- 总则写“为什么需要方案、方案解决什么问题、依据是什么”，不要提前作具体措施承诺。
- 项目基本情况只写与文物保护有关的建设事实，不扩写商业必要性。
- 文物概况以文保单位资料、调查报告、勘探报告和文评报告为来源；无来源处标注【待补充】。
- 保护措施按阶段组织，不要把所有措施堆成一段。
- 每节措施先交代风险，再写控制要求，再写落实方式。

## 高风险表述

以下内容不得无依据生成：

- “无影响”“影响较小”“可忽略”等最终评价。
- “安全距离为 X 米”“外扩 X 米为保护区”。
- “预留保护煤柱”“实施考古发掘”“无需发掘”。
- “经某单位批准”“符合某批复要求”。
- “监测点位坐标为……”。

## 和文评报告的关系

文评报告可提供影响识别、价值评估和空间关系底稿；保护方案必须进一步回答：

- 谁负责落实。
- 什么时候落实。
- 用什么设施、制度、巡查或监测落实。
- 发现异常如何处置。
- 如何形成记录和验收材料。
