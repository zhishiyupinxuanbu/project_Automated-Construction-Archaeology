# {{项目名称}} 项目资料可用信息清单

本表用于回答“项目资料里有哪些信息可以用于文评初稿”。填写时必须保留来源文件，不能只写结论。

| 信息类别 | 可用信息 | 来源文件/路径 | 可用于报告位置 | 可信度 | 风险/待确认 |
| --- | --- | --- | --- | --- | --- |
| 项目基本信息 | 项目名称、建设单位、位置、建设内容、建设规模 |  | 一、总则；三、建设项目规划概况 | 待核验 |  |
| 文物基本信息 | 文物名称、级别、时代、类型、保存现状、价值 |  | 二、建设项目涉及文物概况 | 待核验 |  |
| 保护区划信息 | 保护范围、建设控制地带、缓冲区、管控要求 |  | 二、建设项目涉及文物概况；四、空间关系 | 待核验 | 高风险 |
| 空间关系信息 | 红线、坐标、最近距离、重叠面积、是否进入管控区 |  | 四、项目用地范围与文物空间分布关系 | 待核验 | 高风险 |
| 工程设计信息 | 总平面、建筑高度、体量、材料色彩、功能布局、路线/范围 |  | 三、建设项目规划概况；五、设计合规性评估 | 待核验 |  |
| 建设期信息 | 施工方式、土方开挖、机械作业、临时占地、施工车辆、污染源 |  | 五、建设期影响评估；六、施工期减缓措施 | 待核验 |  |
| 运营期信息 | 运营功能、人流车流、噪声、灯光、油烟、污水、垃圾、巡查管理 |  | 五、运营期影响评估；六、运营期减缓措施 | 待核验 |  |
| 调查勘探信息 | 项目名称、勘探公司、勘探结论、实际完成面积、发现情况、验收意见 |  | 四、勘探工作；五、地下文物影响 | 待核验 | 第四章第三节核心只替换项目名称、勘探公司和勘探结论；实际完成面积按报告原文填入固定句 |
| 审查评审信息 | 文物部门函件、备案告知书、评审意见、专家要求 |  | 一、必要性；七、结论建议；修改说明 | 待核验 | 高风险 |
| 图件照片信息 | 位置图、红线图、正射影像、现状照片、剖面图 |  | 二、文物现状；四、空间关系；正文图件 | 待核验 |  |
| 文评正文图件 | 文物区划图、项目位置图、总平面图、用地范围图、距离图、空间关系图、视线或景观分析图 |  | 二、三、四、五章正文图件 | 待核验 | 不默认从勘探报告取图 |
| 勘探图件 | 勘探范围图、勘探单元图、探孔布设图、剖线或探孔记录图 |  | 五、地下文物影响 | 待核验 | 不插入第四章第三节，仅作内部判断依据 |
| 法规依据 | 法律法规、政策文件、技术规范、评估指南 |  | 一、评估依据；八、支撑法律法规及文件 | 待核验 |  |
