# 文物影响评估章节证据映射规则 v0.1

本文件用于把项目资料映射到文评报告章节，避免正文空泛生成。

## 与两张初稿表的关系

- `项目资料可用信息清单` 先回答资料中有哪些信息可用。
- `信息入文章节映射表` 再回答这些信息进入哪一章哪一节。
- 本文件提供映射规则；真正写初稿时，以项目实际填写后的两张表为直接依据。

## 八章证据映射

| 章节 | 必要证据 | 可选证据 | 缺项处理 |
| --- | --- | --- | --- |
| 一、总则 | 项目名称、位置、建设单位、涉及文物关系、评估必要性、依据文件 | 合同、委托范围、评审要求 | 缺少文物关系或评估必要性时，不进入成稿结论 |
| 二、建设项目涉及文物概况 | 文物名称、级别、位置、材料原文中的保护区划术语及内容、保存现状、调查/发掘/保护工程情况、价值陈述、遗产价值评级 | 文物调查回函、文物调查报告、核查报告、主管部门函件、文物照片、历史沿革、保护工程资料、《世界文化遗产影响评估指南》遗产价值分级量表 | 第二节必须按 `1.遗址基本情况` 至 `6.价值评估` 六个小节展开；缺保护区划或价值评级依据且为必需内容时先阻断，不在正文留占位 |
| 三、建设项目规划概况 | 项目建设内容、规模、总平面、建设必要性、支持性文件、施工方式、运营方式、运营规模、交通组织、污染排放 | 可研报告、可行性研究报告、设计说明、建设方案、项目说明、多方案比选、合同服务内容；支持性文件取得情况只收立项/选址/核准文件或文物调查回函 | 缺设计/运营资料且为必需内容时先阻断；项目建设必要性通常完整摘取可研或建设方案相关章节，缺完整章节且本节必需时阻断，不用开工请示、图纸或项目内容自行概述；支持性文件 PDF 转图片插入时红头正文页必须完整，附件页可不放，不放红线、正射、规划平面图、KML/OVKML、坐标、勘探报告、验收意见书、既有文评或保护方案；人员管理、组织架构、资金安全等一般不入本章 |
| 四、项目用地范围与文物空间分布关系 | 红线、项目坐标、文保点坐标、KML/OVKML、叠合图、距离图、调查函件、所在管控区、选址分析或选址必要性分析原文 | 正射影像、现场踏查图、坐标表、重叠面积量算结果、可研报告、可行性研究报告、建设方案 | 缺最近距离或是否涉及管控区时，不得下确定性结论；面状-面状关系需优先核对双方坐标和叠合结果；线状文物一般重点说明最近距离和已知关系；第二节标题必须为 `（二）【项目名称】选址分析`，内容通常完整摘取可研/建设方案对应章节，不修改、不缩减、不概括，缺原文且必需时阻断 |
| 五、影响分析与评估 | 设计事实、建设期活动、运营期活动、空间关系、文物敏感要素、预填表驱动抽取的具体条文或控制要求 | 量表、矩阵、专家意见 | 证据不足时不编影响等级；必需信息先阻断，非必需信息略过；第五章评估表只作正文分析后的归纳和程度判断，不能替代自然段正文 |
| 六、减缓措施建议 | 与第五章影响路径对应的措施 | 同类成稿措施表达、法规要求、可研/工程方案/施工组织/运营方案中的预防和管理措施 | 措施必须对应影响，不写无来源的技术指标；能引用项目前期方案已有措施时，应写明按方案落实的具体措施 |
| 七、结论及建议 | 前六章已确认的合规性判断、影响程度、关键措施、重评触发条件 | 专家意见、主管部门要求 | 第七章为固定内容，仅作变量替换；项目方案均合规时第一段固定写“基于上述分析……项目设计符合法律法规和相关规范，具有可行性。”；存在须调整后方可合规的内容时，套用“但须……调整后……”固定句式 |
| 八、支撑法律法规及文件 | 第八章固定六项法规政策及条文全文 | 无 | 内容及格式完全固定，不根据项目追加专项法规、保护规划、技术规范或“与本项目关系”分析；只有相关法律法规、政策文件更新版本时才整体更新固定文本 |
| 附表 | 第五章影响评估表、第二章遗产价值分级结论、第六章减缓措施概括 | 觉海寺和正蓝旗成稿附表、多文物对象分表要求、主管部门附表要求 | 固定使用综合大表；结构对照觉海寺和正蓝旗；行列标题不随意改动 |

## 觉海寺样本映射启发

觉海寺成稿显示，建筑改扩建和长城建控地带项目通常需要重点建立以下映射：

- 文物调查函件 -> 第一章必要性、第二章文物身份、第四章空间关系。
- 规划平面图 -> 第三章建设内容、第五章设计合规性。
- 正射影像、红线、坐标、KML -> 第四章位置关系和图件依据。
- 考古调查勘探报告 -> 第四章 `（三）勘探工作` 只提取项目名称、勘探公司和勘探结论并套入固定文本，固定句中的实际完成面积按报告原文填入；不摘写方法、探孔、验收过程，不插图；第五章地下文物影响可另行使用其事实依据。
- 评审意见表 -> 修改闭环、自检清单，不直接替代正文论证。

## 章节写作硬规则

- 第三章必须先交代工程事实，不能在第五章第一次出现关键建设内容。
- 第四章必须写清“项目红线/用地范围”和“文物本体/保护范围/建设控制地带/遗产区/缓冲区等材料原文术语”的关系。点、线、面关系的具体写法见 `references/13-空间关系写作规则.md`。
- 第五章按“设计合规性 -> 建设期 -> 运营期 -> 文物对象综合评估”展开。
- 第五章每节后必须配表：设计合规性评估表、建设期影响评估表、运营期影响评估表、文物对象综合影响评估表。第四节综合表整合建设期和运营期小表，并加入遗产资产价值、总体影响评估和减缓措施字段。
- 第五章第二节、第三节和综合影响评估必须先用自然段写清影响路径、项目事实、对应条文或证据和判断逻辑，再放评估表；不能用评估表替代正文分析。
- 第五章第四节综合影响评估大表前必须固定插入《世界文化遗产影响评估指南》影响评估说明、`assets/16-世界文化遗产影响评估指南-影响评估量表.png`、固定图题和表前承接句。
- 第六章措施要逐条回应第五章影响路径，小节标题可按项目实际问题选择阶段式或问题式组织，不机械套用固定四节。
- 第七章为固定内容，仅替换文物名称、涉及区划、控制对象、控制要求、保护区划表述、项目周边环境表述和项目活动类型，不自由增删段落。
- 第七章第一段由分析模块的合规性判断选择模板：全部合规时用“基于上述分析，本项目建设及运营过程对涉及的【文物名称】的影响相对轻微。项目设计符合法律法规和相关规范，具有可行性。”；须调整后方可合规时用“影响相对轻微。但须……调整后，项目设计符合法律法规和相关规范，具有可行性。”。
- 第七章固定保留现行方案重评触发、系统实施减缓措施、长效监测评估机制、保护方案、监测体系、应急预案、部门沟通、行政审批和施工发现文物停工报告程序。
- 第八章内容及格式完全固定，按 `references/11-固定正文结构与固定内容.md` 的六项法规政策及条文全文写入；不得删减、摘要、改写或追加专项法规，除非相关法律法规、政策文件更新版本。
- 附表固定为综合大表，合并第五章影响评估、第二章价值分级和第六章措施概括；结构对照觉海寺和正蓝旗成稿附表，行列标题固定，不把附表降为可选重复表。
