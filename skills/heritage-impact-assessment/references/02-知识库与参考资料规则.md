# 文物影响评估知识库与参考资料规则 v0.1

## 首要边界

- 本 skill 的默认输入是用户给出的项目资料文件夹。
- 项目事实只从该项目资料、可读取文件或人工确认中提取。
- 旧 `文物影响评估助手/` 包不再作为默认入口，不要让它的目录结构牵引本 skill 的工作流。
- `references/source-originals/` 中的成稿只用于学习文章架构、表达方式和溯源核验，不能替代新项目资料。

## 外部参考资料池

当需要查法规、方法论、报告案例、论文、书籍或文保单位资料，且项目资料与 `references/source-originals/` 不能满足时，可查当前工作包中的已整理参考资料池：

```text
文物影响评估助手/1. 参考资料/
```

该目录是“外部参考资料池”，不是项目输入资料。使用前先读取 `references/04-已整理参考资料目录.md`，再进入对应子目录查找。

## 知识库位置

如用户明确要求调用旧知识库，再查：

```text
文物影响评估助手/2.智能助手资料包/1.文物影响评估知识库/
```

如果该知识库尚未建成，明确说明“知识库未正式展开”。不要因此中断项目成稿流程，应继续用项目资料、`references/source-originals/` 和必要的外部参考资料生成待审核草稿。

## 调用顺序

1. 先扫项目资料文件夹，建立资料目录和证据索引。
2. 再读 `references/source-originals/` 和 `references/05-文评成稿样本与文章架构.md`，确定文章结构和表达方式。
3. 只有在法规、方法论、案例或文保单位资料不足时，才查外部参考资料池或旧知识库。
4. 找不到依据时，输出【参考资料待补充】或【项目资料待补充】，不要混用。
5. 使用参考资料或原始资料时，说明资料文件名或路径，不用模型记忆替代依据。

## 高风险字段

以下字段必须有资料依据或标记待确认：

- 文物保护范围、建设控制地带。
- 项目红线、用地面积、建设规模。
- 最近距离、重叠面积、坐标。
- 文物级别、时代、类型、保存现状。
- 专家意见、文物部门意见、批复文号。
- 影响等级、是否可控、是否需考古发掘。

## 影响识别维度

- 空间占压或重叠。
- 振动、沉降、开挖、爆破、机械扰动。
- 视觉景观和历史风貌。
- 扬尘、废水、固废、噪声、灯光。
- 人员活动、车辆通行、游客或信众误入。
- 管理责任、监测维护、应急处置。
