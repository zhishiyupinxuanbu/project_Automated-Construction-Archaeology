# 公司资料包

本目录按公司存放长期使用的公司信息与人员信息。

每个公司资料包统一使用以下结构：

- `公司信息库/`：公司印章、营业执照、公司资质等公司固定资料。
- `人员信息库/`：按人员姓名集中保存人员附件，同一人员资料只维护一份。
- `人员信息库/具体人员/身份证/`：身份证正面、身份证背面。
- `人员信息库/具体人员/证书/`：岗位证书、结业证书等人员证书。
- `人员信息库/具体人员/劳动合同/`：劳动合同各页，按原文件实际页数保存。
- `人员信息库/具体人员/社保/`：社保证明。
- `*_资料卡.xlsx`：公司基础信息、人员信息、附件清单和版本记录。

`人员信息1`、`人员信息2` 等只作为资料卡中的人员套组名称和工作表名称使用，不再作为长期附件目录。生成器根据所选套组读取人员姓名和顺序，再到 `人员信息库/姓名/资料类别/` 查找附件。

公司资料与人员资料不得混放；脚本工具另放在 `智能生成报告技能资料/工具脚本/公司信息与人员库/`，规则和说明另放在 `智能生成报告技能资料/项目说明/公司信息与人员库/`。
