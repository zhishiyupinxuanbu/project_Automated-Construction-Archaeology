# 三门峡市文物考古勘探有限公司资料包

## 状态

- 人员信息套组：`人员信息1`、`人员信息2`
- 当前状态：已迁移为 `公司信息库` + `人员信息库` 结构，并建立资料卡索引。

## 当前结构

- `公司信息库/`：公司印章、营业执照、公司资质及公司附件清单。
- `人员信息库/`：按人员姓名集中保存身份证、证书、劳动合同和社保；`人员信息1`、`人员信息2` 只作为资料卡中的人员套组名称。
- `三门峡市文物考古勘探有限公司_资料卡.xlsx`：公司基础信息、人员信息套组、公司附件清单、人员附件清单和版本记录。

每个具体人员文件夹下按 `身份证/`、`证书/`、`劳动合同/`、`社保/` 分类存放附件图片。身份证正面为国徽、签发机关、有效期限面；身份证背面为姓名、照片、身份证号面。
