# 文物保护许可申请 / 文物核查请示 - 作战手册

## 适用场景

- 用户说“文物手续”“文物保护许可”“许可申请”“核查请示”等。
- 企业、建设单位、项目单位向文物部门申请核查项目用地范围是否涉及文物保护单位。

## 写作要点

- 默认拟建设单位、项目单位、企业发文，不主动拟文物部门内部转报。
- 正文可保留模板条目结构：开头说明、项目名称、建设规模及内容、建设地点、项目面积、申请核查句、结尾请示句。
- 申请核心是核查建设用地范围内是否涉及文物保护单位，并恳请出具审查意见。
- 面积表述用“项目面积”或“项目用地面积”，不得写“调查面积”“勘探面积”。
- 建设内容尽量依据资料原文，不擅自扩写审批过程和文号。
- 库伦旗项目如由库伦旗文化和旅游局向通辽市文化旅游广电局请示文物核查，优先按库伦旗模板处理：保留红色发文机关标志，标题写“关于对XX项目用地范围内进行文物核查的请示”，正文按“前置核准、批复或备案情况—项目名称—建设规模及主要建设内容—项目地点—申请文物核查—此申请”组织。
- 科右前旗项目办理文物保护许可、文物核查或行政审批申请时，优先按科右前旗模板处理：标题写“关于（单位项目名称）申请用地范围文物保护许可的申请”，受文单位为“科右前旗文化旅游体育局”，正文保持条目式项目情况，依次写项目名称、建设单位、建设地点、项目建设性质、项目法人、建设规模及内容、建设或开采理由，并依据材料写明已取得的文物初选址意见、发改委批复或其他前置意见。
- 锡林郭勒盟、西乌珠穆沁旗项目办理文物保护许可、文物核查或申请办理文物调查事宜时，用户简称“锡盟”“锡林郭勒盟”“西乌”“西乌珠穆沁旗”均优先按锡林郭勒盟西乌珠穆沁旗模板处理：标题保留“申请办理文物调查事宜的请示”口径，受文单位按项目所在地文旅文物主管部门，正文按“建设必要性和项目背景—项目名称—建设地点—主要建设内容—申请核查是否涉及文物保护单位并出具审查意见—特此请示”组织。
- 锡林浩特市项目办理临时道路、临时车辆行驶道路、用地范围查询是否涉及文物时，用户简称“锡林浩特”“锡林浩特市”“锡市”，或项目名称/材料出现“胜利矿区”“胜利煤田”“胜利矿田”且指向锡林浩特市，均优先按锡林浩特市模板处理：受文单位写“锡林浩特市文体旅游广电局”，标题采用“关于申请查询项目用地是否涉及文物的请示”或“关于申请查询项目临时用地是否涉及文物的请示”口径，正文按“项目位于锡林浩特市毛登牧场境内或材料所载地点—项目面积/项目用地面积或路线长度—用地性质—申请核查并出具意见”组织；照片遮挡内容不得猜测。如材料不是临时用地，删除或替换“临时用地”“临时车辆行驶道路”等不符合事实的表述，不得因此回退通用模板。
- 乌兰察布地区项目优先按地区模板处理：标题写“关于XX项目文物核查的请示”，正文按“项目背景和建设情况—申请文物核查并恳请复函—特此请示”组织，新版式默认加红色发文单位抬头和红线，但不改变当地正文和附件口径。
- 杭锦旗项目如处在立项前文物地表调查或文物核查阶段，优先读一次性告知清单附件1；如已取得核准、备案或批复，优先读附件3。附件1和附件3标题口径相近，都可写“XX单位关于XX项目是否涉及文物的申请函”，区别在于附件3正文先写已取得的核准、备案、批复及文号，并将对应文件列为附件。

## 附件

默认附件顺序：

1. 审批机关立项选址核准文件，或材料中的正式前置依据文件名。
2. 企业法人营业执照。
3. 法定代表人身份证复印件。
4. 完整项目名称用地经纬度坐标 Excel 表。
5. 完整项目名称用地范围 KML 格式坐标文件。
6. 完整项目名称用地宗地图。

上述默认附件属于文物核查请示应有附件框架，即使材料未一次性提供齐全，也应先在 Word 附件栏保留完整清单；缺少或不能确认的附件，只在交付说明中提醒用户补充或确认，不在正文或附件栏写“待补充”等占位。

不默认加入考古调查勘探报告、文物保护方案、文物影响评估报告，除非用户要求或当地模板明确要求。

库伦旗模板附件优先按当地样件口径处理，常见顺序为：文物保护许可申请、审批机关立项选址核准文件、企业法人营业执照、法定代表人身份证复印件、工程建设方案、空间范围及用地经纬度坐标 Excel 表、KML 格式坐标文件、用地宗地图、项目联系人、项目概况。生成时应将 PDF 样件中的 `Exce1`、`km1` 识别误差校正为 `Excel`、`KML`。

乌兰察布地区附件优先按当地模板口径处理，常见顺序为项目备案告知书、企业法人营业执照、法定代表人身份证复印件、项目可研报告、空间范围及用地经纬度坐标 Excel 表、KML 格式坐标文件、用地宗地图；公益性项目或材料另有明确清单时，可按实际材料替换第 1、2、4 项。

科右前旗模板附件按当地最小清单列“发改委立项批复、坐标”；当地模板清单内的附件即使材料暂缺也先保留，并在交付说明中提示；如材料另有营业执照、法人身份证、宗地图、KML 等，再按实际材料补列或提示确认。

锡林郭勒盟/西乌珠穆沁旗模板附件优先按当地样件口径处理，常见顺序为：核准批复或其他前置审批文件、可行性研究报告、路径图、项目坐标及 KML 文件、建设单位营业执照、建设单位法定代表人身份证复印件。生成时应将来源样件中的 `km l` 识别/录入误差校正为 `KML`。

锡林浩特市临时用地是否涉及文物请示模板附件优先按照片样件的最小清单处理：临时用地范围及电子版国家2000坐标。模板清单内的附件即使材料暂缺也先保留，并在交付说明中提示；如材料另有 KML、宗地图、营业执照、法人身份证等，再按实际材料补列或提示确认；没有联系人和电话时不写空括号。

## 输出检查

- 标题不得保留高亮或模板提示色。
- 项目名称、发文单位、落款单位一致。
- 受文单位与项目所在地和材料口径匹配。
- 附件名称使用完整项目名称。
- 文物核查请示附件必须保留该文种或地区模板要求的完整清单；材料缺少时提醒补充或确认，不因资料夹暂缺而删除附件项。
- 没有联系人和电话时不写。
